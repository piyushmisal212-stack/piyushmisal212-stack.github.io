Flagship Project

Project TARS

This is my biggest dream project: a fully functional miniature TARS from Interstellar. The goal is to replicate its iconic walking mechanism using high-torque servos controlled via Arduino, and pair it with my AI voice assistant Nexus so it can actually converse like the character in the film.

Nexus uses the Gemini API for intelligence, with a DeepSeek 8B model fallback when credits run low. ElevenLabs handles voice — falling back to Edge TTS when needed. I'm currently finalizing the chassis design and balancing algorithms, while Nexus v3 is simultaneously under development.

It's still a work in progress, but seeing it come to life will absolutely be worth the wait.

AI / Python Development

Nexus AI Assistant

Nexus started as my very first real AI project — and what a journey it's been. I built two basic TTS chatbots first to understand the fundamentals, then leveled up to create Nexus.

It uses Gemini's API as its primary brain, with DeepSeek's 8B parameter model as a smart fallback when credits run low. For voice, ElevenLabs TTS handles output — with Edge TTS as backup. The whole system is gracefully resilient.

I relied on YouTube, GitHub, and Gemini itself to learn and debug throughout. Hundreds of errors, hundreds of solutions — and all of it made the process genuinely adventurous. Nexus v3 is now in development.
